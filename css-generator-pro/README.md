# 🎨 CSS Generator Pro

CSS Generator Pro is a modern and responsive web application that helps developers and designers create custom CSS styles visually. The tool provides real-time previews and automatically generates ready-to-use CSS code for box shadows, border radius, gradients, and glassmorphism effects.

## 🚀 Features

* Box Shadow Generator
* Border Radius Generator
* CSS Gradient Generator
* Glassmorphism Effect Generator
* Live Preview Panel
* Real-Time CSS Code Generation
* Copy CSS to Clipboard
* Reset Controls
* Dark / Light Mode
* Responsive Design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📖 How To Use

1. Adjust the Box Shadow sliders.
2. Set the Border Radius value.
3. Choose Gradient colors and angle.
4. Customize Glassmorphism settings.
5. View changes instantly in the Live Preview box.
6. Copy the generated CSS code with one click.
7. Reset controls anytime to default settings.
